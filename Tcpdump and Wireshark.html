Overview

In this project, I explore the powerful combination of tcpdump and Wireshark for network traffic analysis. Tcpdump, a command-line packet capture tool, allows for efficient capture of network data, while Wireshark provides an intuitive graphical interface for in-depth analysis and visualization of that captured data. This project demonstrates my proficiency in using these essential tools to dissect network traffic, identify potential security issues, and troubleshoot network problems.
